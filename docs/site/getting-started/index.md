# Getting Started

Follow this path:

1. Install the package or sync a local checkout with Python 3.13.
2. Run the in-memory [quickstart](../quickstart.md).
3. Read the [beta scope](../concepts/beta-scope.md).
4. Use the [backtesting guide](../guides/backtesting.md) for `source` and
   materialized series runs.
5. Use the [public API reference](../reference/public-api.md) when choosing
   imports.

The first beta is intentionally narrow. Start with in-memory or local historical
data before using exchange-backed historical loading.
